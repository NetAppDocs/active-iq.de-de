= Bewertung einer Technologieaktualisierung
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Wenn Sie herausfinden möchten, ob eine Technologieaktualisierung für Ihren Vertrag für den technischen Support oder für Ihre Hardware empfohlen wird, können Sie die Option zur Technologieaktualisierung nutzen.


NOTE: Sie können auf diese Funktion entweder über die Lebenszyklusplanung (*Speicher* > *Lebenszyklusplanung*) oder über Digital Advisor (*Dashboard > Planungs-Widget > Tech Refresh*) zugreifen.

Weitere Informationen zu dieser Funktion finden Sie unterlink:https://docs.netapp.com/us-en/bluexp-economic-efficiency/use/tech-refresh.html["Bewertung einer Technologieaktualisierung"] in der Konsolendokumentation.
