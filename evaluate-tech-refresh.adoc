= Prüfen Sie, ob eine Systemaktualisierung für Ihr Speichersystem im Digital Advisor empfohlen wird
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Wenn Sie herausfinden möchten, ob eine Systemaktualisierung für Ihren Tech Support-Vertrag oder Ihre Hardware empfohlen wird, können Sie die Option „Systemaktualisierung“ verwenden.

Sie können auf diese Funktion entweder über die Lebenszyklusplanung (*Speicher* > *Lebenszyklusplanung*) oder über Digital Advisor (*Dashboard > Planungs-Widget > Systemaktualisierung*) zugreifen.

Weitere Informationen zu dieser Funktion finden Sie link:https://docs.netapp.com/us-en/bluexp-economic-efficiency/use/tech-refresh.html["Eine Systemaktualisierung evaluieren"] in der Dokumentation der Konsole.
